# Google / Alphabet H1 2026 Strategic Performance Dashboard

**Author:** Rahima Akter Diti
**Role:** Lead Data Analyst

This repository contains a strategic analysis and interactive dashboard for Google/Alphabet's H1 2026 performance. It integrates official Q1 2026 data with modeled H1 guidance to provide a holistic view of the company's financial and operational health, specifically focusing on Customer Lifetime Value (CLV).

## Dashboard & Analysis Highlights

* **Financial Performance:** Visualization of total H1 Revenue, Contribution Margins, and growth trends across global segments.
* **Strategic Focus:** Analysis of Search advertising dominance, YouTube churn-margin tension, and Cloud infrastructure CAC payback periods.
* **Interactive Data:** Dynamic charts allowing users to explore segment-level performance, churn risks, and predicted CLV outcomes.

## Tools & Techniques

* **Financial Modeling:** Microsoft Excel (Q1 actuals + H1 projections based on industry WACC).
* **Data Visualization:** HTML, CSS, and Chart.js for interactive, real-time reporting.

## Disclaimer

This analysis is based on publicly available Q1 2026 financial data and independent modeling. It does not represent internal Google/Alphabet operational data or official company forecasts.
